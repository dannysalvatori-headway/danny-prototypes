Test 1: Relevance-First Hero. Multi-dimensional H1 echoes the full search intent (condition + location + insurance). Personalized CTA replaces generic "Get started." Secondary "browse below" CTA for high-intent visitors. Runs on ALL non-brand SEM.

Test 2: Category Sections + Card Badges. Builds on Test 1 hero. Replaces flat provider grid with search-term-relevant category sections (like homepage category carousels but structured for SEM intent). Each section has a descriptive header reinforcing the search term. Cards get relevance badges (✓ Aetna, ✓ Anxiety specialist) and availability signals. Inspired by homepage category sections test (+1.47% match rate). Runs on ALL non-brand SEM.
2b
Test 2b: Sections + Explore Carousels. Same as Test 2, but adds homepage-style browse carousels below the primary sections. First section = exact match to search intent (same as Test 2). Below that = "Explore more" carousels for adjacent categories (e.g., anxiety user also sees Depression, Trauma, Psychiatrists). Each carousel is carrier-filtered when insurance is known. Serves the Exploratory persona and addresses the "not sure yet" segment that Test 2 doesn't help. Inspired by homepage category sections (+1.47% match, carousel users click profile intake CTA at 13x rate).
