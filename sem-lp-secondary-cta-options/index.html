SEM LP — Secondary CTA Placement Options

5 options for surfacing “browse providers” as a secondary path without sandwiching it between the primary CTA and cost banner. All shown on mobile (where placement is most constrained). Each tab highlights only the changed element.

Placement
Option 1: Paired with CTA. Text link directly under “Get matched” button: “or browse 4,319 therapists near you ↓”. Lives inside the form group so it reads as an alternative to the same decision, not a separate section. Visually light — muted text with underline, not a button.
2
Option 2: Clickable Proof Strip. The “4,319 therapists” item in the proof strip becomes an anchor link that scrolls to the provider grid. Subtle underline + “↓” indicator. No new element added — zero visual weight. Downside: low discoverability, easy to miss.
3
Option 3: Sticky Footer. The browse CTA only appears in the sticky footer after the user scrolls past the hero. Footer shows: text + “Get matched” (primary) + “Browse ↓” (text link). Contextually smart — the user already passed the hero without clicking, so they’re in browse mode. No change to the hero at all.
4
Option 4: H1 Subtitle. Small text link below the H1, above the proof strip: “or browse 4,319 therapists near you ↓”. Reads as a continuation of the headline. Signals there’s content below without competing with the form. Downside: adds vertical distance between H1 and form fields.
5
Option 5: In Cost Banner. Adds a second line to the cost banner: “Browse therapists accepting your insurance ↓”. Reframes the banner from single-purpose cost signal to a dual-purpose element. The banner is already clickable-looking (card + arrow). Downside: may dilute the ambient cost signal that drives +3.7% lift.
